# claude Provider

The Claude provider allows you to manage [Anthropic Claude](https://www.anthropic.com/) resources using the [Admin API](https://docs.anthropic.com/en/docs/administration/administration-api).

## Authentication

The provider requires an Admin API key. You can provide it in two ways:

1. Set the `api_key` attribute in the provider configuration.
2. Set the `ANTHROPIC_ADMIN_API_KEY` environment variable.

~> **Note:** If both are set, the provider configuration takes precedence.

## Example Usage

```terraform
provider "claude" {
  # Can also be set via ANTHROPIC_ADMIN_API_KEY environment variable.
  api_key = var.anthropic_admin_api_key

  # base_url = "https://api.anthropic.com"  # default
}
```
